### 🎠 [Carousel Studio](file:///Users/cortex/.gemini/antigravity/brain/c071ee7e-8d36-4b7e-91cf-89ae722fce8e/v2_architecture.md)
- **New API:** `POST /api/v2/posts/{id}/carousel`
- **New Model:** `CarouselAsset` (slides JSON, PDF URL, LinkedIn URN)
- **New Microservice:** Carousel Renderer (Puppeteer/WeasyPrint) → PDF → GCS
- **AI Engine:** New endpoint generates 7-slide JSON outline
- **Publishing:** LinkedIn 3-step Document Upload API via write-flow OAuth

### ⚡ Virality Scoring Engine
- **New API:** `POST /api/v2/posts/{id}/score`
- **Extends:** `Post` model with `virality_score`, `hook_alternatives`, `actual_engagement_rate`
- **AI Engine:** Takes user's top 20 posts as training examples, outputs 1–100 score + 3 hooks
- **Background worker:** `engagement_sync_worker` polls live LinkedIn metrics every 6h → closes the DPO feedback loop

### 🌡️ Posting Time Heatmap
- **New API:** `GET /api/v2/analytics/heatmap`
- **Pure SQL query** on existing `posts` table — no new models needed
- **Frontend:** CSS heat overlay on calendar grid, drag-to-red-slot warning toast
- **Works Day 1** with LinkedIn global benchmarks; personalizes after 5+ published posts

### 🤖 Smart Fill Calendar
- **New API:** `POST /api/v2/calendar/smart-fill`
- **New Service:** Calls heatmap for best slots + AI Engine for week plan + bulk-creates drafts
- **AI Engine:** Generates N posts with pillar assignment, format, hook, body, CTA — calibrated to user's voice via top-post examples
- **Frontend:** Drawer panel → pillars + frequency → AI-filled calendar with draggable cards
